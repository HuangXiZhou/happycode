# happyday
>Happy variable name translation

## Usage

```bash
  Usage: happycode [options]

  变量名翻译（英 => 中）


  Options:

    <keyword>           翻译单词（中）

    -E, --en <keyword>  翻译单词（英）

    -h, --help          output usage information


  Example:

$ happycode 变量
$ happycode --en variable
```
## Installation

* Install Node.js and npm
* Run npm install -g happycode
